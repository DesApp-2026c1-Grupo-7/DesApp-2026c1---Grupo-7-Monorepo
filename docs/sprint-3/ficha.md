# Sprint 3 - 21/05 -> 04/06

**Objetivo:** implementar sesiones de estudio colaborativo (punto 6) sobre la base social del Sprint 2.

## Alcance

- Crear sesion con materia, tema, tipo, fecha, duracion, cupos, descripcion y aprobacion requerida.
- Sesion virtual con link o presencial con ubicacion.
- Visibilidad segun privacidad del creador.
- Filtros de busqueda.
- Inscripcion de estudiantes.
- Aprobacion manual si corresponde.
- Email de confirmacion.
- Editar/cancelar sesion y notificar inscriptos.
- Recordatorio automatico 24h antes.

## Definition of Done

- [ ] Un estudiante crea sesion virtual/presencial.
- [ ] Otro estudiante puede inscribirse.
- [ ] Si requiere aprobacion, el creador aprueba/rechaza.
- [ ] Cancelar sesion notifica a inscriptos.
- [ ] Existe recordatorio 24h antes.
- [ ] CI verde y tests de flujo principal.
